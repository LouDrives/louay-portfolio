Platforme Étudiant – AI OCR System

Ministry of Finance · Python · Tesseract OCR · Streamlit · Cloud Vision

Developed an AI‑powered platform to read and verify Moroccan CIN and Baccalaureate documents. Designed the OCR pipeline, data validation layer and user interface for real‑time document checking.

The prototype automatically extracts key fields from scanned documents, flags inconsistencies and assists human operators in making eligibility decisions faster and more reliably.

OCR Computer Vision Python Streamlit

Automated Feedback Analyzer

Academic Project · Python · NLP · Word2Vec · TF‑IDF · Streamlit

Designed a web app that processes large volumes of textual feedback to extract sentiment, topics and concise summaries. Combined classical NLP (TF‑IDF) with distributed representations (Word2Vec) to better capture semantic relationships.

The tool provides dashboards that help stakeholders quickly understand what users are saying, what themes dominate, and where to focus improvements.
